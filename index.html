Các bước đẩy code lên github

1. Tạo 1 repository trên github: tạo trên web hoặc tạo bằng git bash 2. Clone repository về máy tính: git clone 3. Thêm file vào repository: git add 4. Commit file: git commit -m "Nội dung commit" 5. Đẩy code lên github: git push origin Những câu lệnh khá
